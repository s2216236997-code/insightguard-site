AI governance that behaves like infrastructure.

Insight Guard is a runtime governance and enforcement layer for AI systems: deterministic decisions, stable reason codes, and auditable outputs — without storing raw user data or interfering with core business.

Single Guard Endpoint

POST /v1/guard returns allow | block | cooldown | no_op with a stable reason_code and audit_id.

Determinism Layer

Frozen behavior_version. Behavior changes only via new version releases — no silent drift.

Fail-safe by Design

Configurable fail-open / fail-closed. Guard outages should never take down your core product.
